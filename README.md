## âš™ï¸ How The Animation Works

-   The original text is stored when the page loads.\
-   On mouse hover:
    -   A timed interval starts running.\
    -   Each letter is replaced with a random character from a
        predefined alphabet string.\
    -   Letters gradually resolve back to the original text.\
    -   Once all letters are restored, the animation stops
        automatically.
